# MERN-Project1-hostech
A full-stack web application built with the MERN stack (MongoDB, Express.js, React.js, Node.js). It features secure user authentication using JWT and bcrypt, Redux for state management, Axios for API calls, protected routes, and a user profile page. Demonstrates modern full-stack development and authentication workflow.
# MERN Authentication App

Tech Stack:
- React
- Node.js
- Express
- MongoDB
- Redux Toolkit
- JWT Authentication

Features:
- User Signup
- Login Authentication
- Profile Update
- Image Upload
- Password Change
